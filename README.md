# CAMPS Member Matching Program & Supply Chain Engine

An advanced, dual-track capability and operational match engine dashboard built for CAMPS (Center for Advanced Manufacturing Puget Sound).

## Features
- **Track A (Physical Production Capacity)**: Filter by machinery types (e.g. 5-Axis CNC, Sheet Metal Fab), materials, and certifications.
- **Track B (Technology & Automation)**: Filter by software solutions and map traditional manufacturers experiencing shop-floor bottlenecks to tech-focused members.
- **Shared / Excess Capacity Directory**: Spot underutilized machine shifts (e.g., CMM inspection, laser rigs, autoclaves) to co-use assets and prevent redundant capital purchases.
- **Interactive Mapping**: Geographic visualization of members across Washington State with color-coded markers for tracks and shared capacities.
- **Shop-Floor Pain-Point Matcher**: Natural language presets mapping operational problems directly to Track B technology providers.

## Deployment
This app is designed to run on **Streamlit Community Cloud** directly connected to this repository.
